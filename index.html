The Rules

  • Listen to all the songs.
  • Rank your Top 5 favorite songs.

Tie-breaker Round

If two or more songs receive the same number of points, or if one or more songs receive 0 points, an additional voting round will be held.

  • The second round will include only new songs, with the number of new songs matching the number of tied or zero-point songs.
  • If there are still ties or songs with 0 points after the second round, a third voting round will be held following the same procedure.
